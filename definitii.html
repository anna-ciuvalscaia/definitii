Definitii HTML si CSS

HTML HyperText Markup Language (HTML) este un limbaj de marcare utilizat pentru crearea paginilor web ce pot fi afișate într-un browser (sau navigator). Scopul HTML este mai degrabă prezentarea informațiilor – paragrafe, fonturi, tabele ș.a.m.d. – decât descrierea semanticii documentului.
small tag pereche -Eticheta definește textul mai mici (cum ar fi drepturile de autor și alte observații secundare).
sub tag pereche - Eticheta definește textul subscript. Textul subscris apare cu jumătate de caracter sub linia normală și este redat uneori cu un font mai mic. Textul subscript poate fi folosit pentru formule chimice, cum ar fi H 2 O.
sup tag pereche - Eticheta definește textul exponenți. Textul suprascript apare cu jumătate de caracter peste linia normală și este redat uneori într-un font mai mic. Textul suprascript poate fi utilizat pentru notele de subsol, cum ar fi WWW [1] .
img src="..." alt="..." tag-ul "img" este folosit pentru a încorpora o imagine într - o pagină HTML.
Tag - ul are două atribute obligatorii:
  • src - Specifică calea către imagine
  • alt - Specifică un text alternativ pentru imagine, dacă imaginea nu poate fi afișată din anumite motive
ol tag pereche - Eticheta definește o listă ordonată. O listă ordonată poate fi numerică sau alfabetică.
ul tag pereche - Eticheta definește o lista neordonata (bulleted). tag-ul "ul" se utilizeaza împreună cu tag "li" tag - ul pentru a crea liste neordonate.
title - este un tag -pereche, care defineste titlul documentului. Titlul trebuie să fie numai text și este afișat în bara de titlu a browserului sau în fila paginii.
html lang="en" este un atribut in interiorul etichetei pentru a declara limba paginii web. Acest lucru este menit sa ajute motoarele de cautare si browserele.
header Elementul reprezintă un container pentru conținutul introductiv sau un set de link - uri de navigație.
Un "header" element conține de obicei:
  • unul sau mai multe elemente de titlu (h1 - h6)
  • sigla sau pictograma
  • informații despre autor
Notă: Puteți avea mai multe "header" elemente într-un singur document HTML. Cu toate acestea, "header" nu poate fi plasat într-un "footer", "address" sau într-un alt "header" element.
footer Eticheta definește un subsol pentru un document sau o secțiune.
Un "footer" element conține de obicei:
  • informații despre autor
  • informatii despre copyright
  • informatii de contact
  • sitemap
  • înapoi la linkurile de sus
  • documente relatate
Puteți avea mai multe "footer" elemente într-un singur document.
section tag pereche - Eticheta definește o secțiune într - un document.
main Eticheta specifică conținutul principal al unui document.
Conținutul din interiorul "main" elementului ar trebui să fie unic documentului. Nu ar trebui să conțină niciun conținut care se repetă pe documente precum barele laterale, link-uri de navigație, informații privind drepturile de autor, logo-urile site-ului și formularele de căutare.
Notă: Nu trebuie să existe mai mult de un "main" element într-un document. "main"-Elementul nu trebuie să fie un descendent al unui "article, section, footer, header sau nav" Element.
article Eticheta specifică de conținut independent, autonom. Un articol ar trebui să aibă sens singur și ar trebui să fie posibil să îl distribuie independent de restul site-ului. Surse potențiale pentru "article" element:
  • Postare pe forum
  • Postare pe blog
  • Poveste noua

Notă:"article" Elementul nu face nimic special ca într - un browser. Cu toate acestea, puteți utiliza CSS pentru stilul "article" elementului (a se vedea exemplul de mai jos).
figure Eticheta specifică de conținut autonom, cum ar fi ilustrații, diagrame, fotografii, liste de cod, etc.
În timp ce conținutul "figure" elementului este legat de fluxul principal, poziția acestuia este independentă de fluxul principal, iar dacă este eliminat, acesta nu ar trebui să afecteze fluxul documentului.
Sfat: "figcaption" - elementul este folosit pentru a adăuga o legendă pentru "figure" elementul.
nav Eticheta definește un set de link - uri de navigare.
Observați că NU toate linkurile unui document ar trebui să fie în interiorul unui "nav" element. Elementul "nav" este destinat numai pentru blocul major al link - uri de navigare.
SEO SEO este o știință relativ nouă (dacă considerăm că și internetul este destul de nou 🙂 ) și reprezintă o combinație dintre diferite domenii mai vechi (marketing, scriere texte, diferite tehnici de atragere a clienților etc.).
În SEO se lucrează cu conținutul unui site și cu modul în care este perceput acesta atât de către utilizator, cât și de motoarele de căutare, precum Google.
schema org Schema.org este o activitate de colaborare, comunitară, cu misiunea de a crea, menține și promova scheme de date structurate pe internet, pe pagini web, în ​​mesaje de email și nu numai.
Vocabularul Schema.org poate fi utilizat cu multe codificări diferite, inclusiv RDFa, Microdata și JSON-LD. Aceste vocabulare acoperă entitățile, relațiile dintre entități și acțiuni și pot fi extinse cu ușurință printr-un model de extensie bine documentat. Peste 10 milioane de site-uri folosesc Schema.org pentru a marca paginile web și mesajele de e-mail. Multe aplicații de la Google, Microsoft, Pinterest, Yandex și altele folosesc deja aceste vocabulare pentru a oferi experiențe bogate în energie.
Fondată de Google, Microsoft, Yahoo și Yandex, vocabularile Schema.org sunt dezvoltate printr-un proces comunitar deschis , folosind lista de distribuție public-schemaorg@w3.org și prin GitHub .
Un vocabular împărtășit face mai ușor pentru webmasteri și dezvoltatori să decidă asupra unei scheme și să obțină beneficiul maxim pentru eforturile lor. În acest spirit, fondatorii, împreună cu comunitatea mai mare, s-au reunit - pentru a oferi o colecție comună de scheme.
microformats Microformatele sunt modele mici de HTML pentru a reprezenta lucruri publicate în mod obișnuit, cum ar fi persoane , evenimente , postări pe blog , recenzii și etichete în paginile web.
Microformatele sunt cel mai rapid și simplu mod de a furniza o API informațiilor de pe site-ul dvs. web. Vedeți ce altceva puteți face cu microformate .
CSS Css-ul este un limbaj de stilizare al elementelor html, al tagurilor html. Denumirea CSS provine din expresia Cascading Style Sheets. In Web Design-ul modern, pentru stilizarea paginilor web se foloseste numai CSS. Acest lucru inseamna ca de la culoarea literelor si a backgroundului pana si la pozitionarea elementelor de pe o pagina web, totul este stilizat prin CSS. Stilurile folosite pe o pagina pot fi incorporate in pagina respectiva sau pot fi chemate din fisiere externe, fisiere css.
